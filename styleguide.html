Navigation Button

CSS selector:

.nav-main-item, box-item

Sample code:

<nav class="nav-main box">
<a href="#" class="nav-main-item box-item">Shop</a>
</nav>

Rendered element:

Shop

In-text link

CSS selector:

.textlink

Sample code:

<a href="#" class="textlink">Parachute Hammock</a>

Rendered element:

Parachute Hammock

Submit/Reset Buttons

CSS selector:

.button

Sample code:

<a class="button" href="#">Submit</a>
<a class="button" href="#">Reset</a>

Rendered element:

Submit Reset
